# **@lerna/info**

> 打印本地环境信息

安装[lerna](https://www.npmjs.com/package/lerna)以访问**lerna** 命令行界面。

## Usage

这 **info** 打印被证明是有用的本地环境信息，尤其是在提交错误报告时。

**lerna info**
